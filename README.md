# pilas-generador-windows
Scrips para generar el cargador de pilas-engine para windows.

## Instrucciones

Es importante instalar git para windows, y lanzar los scrips en
orden, usando la opcion "git shell" que aparece en el menú
contextual de windows luego de instalar git.

Luego se tiene que descomprimir el archivo "extras\pyinstaller-pyinstaller-953f6e3.zip" para que lo puedan utilizar los scripts.

Hay mas detalles sobre este metodo de empaquetado en
el siguiente artículo:

 - http://www.examplelab.com.ar/haciendo-un-cargador-de-juegos-python/
